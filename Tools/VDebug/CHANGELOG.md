# CHANGELOG - VDebug (Tool)
# ==========================
# V Rising development debugging toolkit
# ==========================

## [0.1.0] - 2026-01-15
----------------------------------------

ADDED:
  - VDebug plugin with asset dumping and UI inspection
  - UI Inspector Service
    - Layout information display
    - Sibling navigation
    - Style info extraction
  - Export functionality for UI component details
  - Font Loading Service
    - Asset bundle support
    - In-game resource loading
  - IMGUI debug panel (F9 to toggle)
  - RectTransform live controls
    - Anchors, offsets, pivot, sizeDelta
  - Layout Group controls
    - Spacing, padding, cellSize, constraint
  - Text controls
    - fontSize, lineSpacing, alignment
  - Runtime logging with "Dump Layout" button
