# Blood Legacies

The legacy system tracks blood type mastery and provides defensive/utility bonuses.

## Overview

- Gain legacy levels in blood types when feeding
- Choose stats to enhance per blood type
- Stats scale with your legacy level
- Class synergies multiply certain stats

---

## How It Works

### Gaining Legacy

Legacy experience is earned when you:
- **Execute** enemies with the blood type
- **Complete** feeding on enemies

<Callout type="info">
  You must finish the feed or execute to gain legacy XP!
</Callout>

### Blood Types

Each blood type has its own legacy track:
- Warrior, Rogue, Brute
- Scholar, Creature, Worker
- Draculin, Mutant

---

## Stat Bonuses

Pick stats to enhance when using each blood type:

| Stat | Base Cap | Description |
|------|----------|-------------|
| Healing Received | 0.15 | Increased healing |
| Damage Reduction | 0.05 | Flat damage reduction |
| Physical Resistance | 0.10 | Physical damage resist |
| Spell Resistance | 0.10 | Spell damage resist |
| Resource Yield | 0.25 | Bonus resource gathering |
| Reduced Blood Drain | 0.50 | Slower blood consumption |
| Spell Cooldown Recovery | 0.10 | Faster spell cooldowns |
| Weapon Cooldown Recovery | 0.10 | Faster weapon cooldowns |
| Ultimate Cooldown Recovery | 0.20 | Faster ultimate cooldowns |
| Minion Damage | 0.25 | Bonus minion damage |
| Ability Attack Speed | 0.10 | Faster ability animations |
| Corruption Damage Reduction | 0.10 | Resist corruption damage |

### Class Synergies

Classes enhance specific blood stats with a multiplier:
- Blood Knight: DamageReduction, ReducedBloodDrain, WeaponCooldown, AbilitySpeed
- Vampire Lord: DamageReduction, SpellResist, UltimateCooldown, CorruptionReduce
- And so on...

## Chat Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| `.blood get [Type]` | `.bl get` | View legacy progress |
| `.blood log` | `.bl log` | Toggle gains logging |
| `.blood list` | `.bl l` | List blood types |
| `.blood liststats` | `.bl lst` | List available stats |
| `.blood choosestat [Blood] [Stat]` | `.bl cst` | Select/toggle bonus stat |
| `.blood resetstats` | `.bl rst` | Reset stats for current blood |

<Callout type="tip">
  Using `.bl cst` on an already-selected stat will **remove** it, allowing you to change choices without a full reset.
</Callout>

## Configuration

### Core Settings

| Option | Default | Description |
|--------|---------|-------------|
| `LegacySystem` | `false` | Enable/disable legacies |
| `MaxBloodLevel` | `100` | Maximum legacy level |
| `LegacyStatChoices` | `3` | Stats you can pick (max 3) |

### XP Multipliers

| Option | Default | Description |
|--------|---------|-------------|
| `UnitLegacyMultiplier` | `1` | XP from regular units |
| `VBloodLegacyMultiplier` | `5` | XP from VBloods |

### Stat Caps

Each stat has a configurable cap. See [Configuration Reference](/reference/config) for the full list.

### Reset Cost

| Option | Default | Description |
|--------|---------|-------------|
| `ResetLegacyItem` | `576389135` | Item PrefabGUID for reset |
| `ResetLegacyItemQuantity` | `500` | Cost to reset stats |
