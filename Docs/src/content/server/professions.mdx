# Professions

The profession system provides bonuses for gathering and crafting activities.

## Overview

- Level up professions through related activities
- Higher levels = better bonuses
- Eight profession types to master
- Extra recipes when enabled

---

## Profession Types

### Gathering Professions

| Profession | Activity | Bonus |
|------------|----------|-------|
| **Mining** | Breaking ore | Bonus resources + gold ore |
| **Woodcutting** | Chopping trees | Bonus resources + saplings |
| **Harvesting** | Gathering plants | Bonus resources + seeds |
| **Fishing** | Catching fish | Extra fish every 20 levels |

### Crafting Professions

| Profession | Activity | Bonus |
|------------|----------|-------|
| **Blacksmithing** | Weapon/armor crafting | +10% base stats, x2 durability |
| **Tailoring** | Cloth crafting | +10% base stats, x2 durability |
| **Enchanting** | Jewelry crafting | +10% base stats, x2 durability |
| **Alchemy** | Potion brewing | x2 effectiveness & duration |

<Callout type="info">
  Alchemy bonuses don't increase Holy Potion effectiveness, but duration is extended.
</Callout>

---

## How Bonuses Work

### Gathering Bonuses

When you break a resource node, profession level adds:
1. **Base bonus** — Extra of what you gathered
2. **Profession drops** — Gold ore (Mining), saplings (Wood), seeds (Harvest)

### Crafting Bonuses

When you craft equipment:
- **Base stat increase** — Up to 10% at max level
- **Durability increase** — Up to 2x at max level

## Chat Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| `.profession get [Type]` | `.prof get` | View profession level |
| `.profession list` | `.prof l` | List all professions |
| `.profession log` | `.prof log` | Toggle XP logging |
| `.profession set [Player] [Type] [Level]` | `.prof set` | (Admin) Set level |

## Configuration

### Core Settings

| Option | Default | Description |
|--------|---------|-------------|
| `ProfessionSystem` | `false` | Enable/disable professions |
| `ProfessionFactor` | `1` | XP multiplier |
| `DisabledProfessions` | `""` | Comma-separated list to disable |
| `ExtraRecipes` | `false` | Enable extra recipes |
