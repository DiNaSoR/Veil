# Prestige & Exoform

The prestige system provides an endgame progression loop with powerful rewards.

## Overview

- Reset progress to earn permanent bonuses
- Multiple prestige types (Leveling, Expertise, Legacies)
- Exoform transformations for advanced players
- Leaderboards to compete with others

---

## How Prestige Works

### Basic Loop

1. **Reach max level** in a system (Leveling, Expertise, or Legacy)
2. **Prestige** to reset that progress
3. **Gain bonuses** that persist across resets
4. **Repeat** for increasing power

### Prestige Effects

Each prestige level provides:
- Permanent buffs (configurable)
- Increased stat caps (Expertise/Legacy)
- Access to class spells (Leveling)

---

## Prestige Types

### Leveling Prestige

Reset your player level to gain:
- Prestige buffs
- Class spell unlocks
- Increased Expertise/Legacy rate gains
- Leaderboard ranking

<Callout type="warning">
  Prestiging resets accumulated Rested XP!
</Callout>

### Expertise Prestige

Reset a weapon's expertise to gain:
- Higher stat caps for that weapon
- Reduced rate gains
- Mastery demonstration

### Legacy Prestige

Reset a blood type's legacy to gain:
- Higher stat caps for that blood
- Reduced rate gains
- Mastery demonstration

## Exoform

After maxing normal prestiges, unlock Exoform:

### Requirements
- Complete all normal prestige levels
- Enable `ExoPrestiging` in config

### Features
- **100 Exo Prestige levels** available
- **Exo rewards** — Items on each prestige
- **Exoform transformation** — Powerful vampire forms
- **Primal Echoes** — Purchase VBlood familiars

### Transforming

Toggle exoform transformation:
```
.prestige exoform  → Toggle taunt-to-exoform
```

Taunt to enter exoform when:
- You have stored energy (minimum 15s)
- Energy drains during transformation
- Duration scales with exo prestiges

### Exoform Selection

Choose between available forms:
```
.prestige sf EvolvedVampire
.prestige sf CorruptedSerpent
```

---

## Primal Echoes

With `PrimalEchoes` enabled, spend exo prestige rewards to unlock VBlood familiars:

```
.fam echoes [VBloodName]
```

Cost scales by VBlood tier:
- Regular VBloods = lower cost
- Shard bearers = highest cost (25x base)

---

## Leaderboards

When enabled, track top players by prestige:

```
.prestige lb [Type]  → View leaderboard
```

Admins can exclude accounts:
```
.prestige ignore [Player]  → Toggle leaderboard exclusion
```

## Configuration

### Core Settings

| Option | Default | Description |
|--------|---------|-------------|
| `PrestigeSystem` | `false` | Enable prestige |
| `MaxLevelingPrestiges` | `10` | Max leveling prestiges |
| `MaxExpertisePrestiges` | `10` | Max expertise prestiges |
| `MaxLegacyPrestiges` | `10` | Max legacy prestiges |

### Prestige Buffs

| Option | Default | Description |
|--------|---------|-------------|
| `PrestigeBuffs` | `"1504279833,0,..."` | Buff PrefabGUIDs per level |
| `PrestigeLevelsToUnlockClassSpells` | `"0,1,2,3,4,5"` | Spell unlock levels |

### Rate Modifiers

| Option | Default | Description |
|--------|---------|-------------|
| `LevelingPrestigeReducer` | `0.05` | XP reduction per prestige |
| `PrestigeRatesReducer` | `0.10` | Expertise/Legacy rate reduction |
| `PrestigeStatMultiplier` | `0.10` | Stat cap increase per prestige |
| `PrestigeRateMultiplier` | `0.10` | Rate increase per leveling prestige |

### Exoform Settings

| Option | Default | Description |
|--------|---------|-------------|
| `ExoPrestiging` | `false` | Enable exo prestiges |
| `ExoPrestigeReward` | `28358550` | Reward item PrefabGUID |
| `ExoPrestigeRewardQuantity` | `500` | Reward amount |
| `TrueImmortal` | `false` | Immortal blood during exoform |

### Leaderboard

| Option | Default | Description |
|--------|---------|-------------|
| `Leaderboard` | `true` | Enable leaderboards |
