# Classes

The class system provides unique playstyles with on-hit effects and shift spells.

## Overview

- Select a class to gain themed abilities
- Each class has weapon and blood stat synergies
- On-hit effects proc debuffs when dealing damage
- Shift spells unlock through prestige levels

## On-Hit Effects

When dealing damage, you have a chance to proc your class's debuff:

1. **First effect** â€” Initial debuff (Leech, Static, Chill, etc.)
2. **Second effect** â€” If target already has first debuff, apply a follow-up effect (debuff or self-buff depending on class)

The proc chance is configurable (default 7.5%).

## Stat Synergies

Each class enhances specific stats from Expertise and Legacies:

### Blood Knight
- **Weapon:** MaxHealth, PrimaryAttackSpeed, PrimaryLifeLeech, PhysicalPower
- **Blood:** DamageReduction, ReducedBloodDrain, WeaponCooldownRecovery, AbilityAttackSpeed

### Demon Hunter
- **Weapon:** MovementSpeed, PrimaryAttackSpeed, PhysicalCritChance, PhysicalCritDamage
- **Blood:** PhysicalResistance, ReducedBloodDrain, WeaponCooldownRecovery, MinionDamage

### Vampire Lord
- **Weapon:** MaxHealth, SpellLifeLeech, PhysicalPower, SpellPower
- **Blood:** DamageReduction, SpellResistance, UltimateCooldownRecovery, CorruptionDamageReduction

### Shadow Blade
- **Weapon:** MovementSpeed, PrimaryAttackSpeed, PhysicalPower, PhysicalCritDamage
- **Blood:** SpellResistance, ReducedBloodDrain, WeaponCooldownRecovery, AbilityAttackSpeed

### Arcane Sorcerer
- **Weapon:** SpellLifeLeech, SpellPower, SpellCritChance, SpellCritDamage
- **Blood:** HealingReceived, SpellCooldownRecovery, UltimateCooldownRecovery, AbilityAttackSpeed

### Death Mage
- **Weapon:** MaxHealth, SpellLifeLeech, SpellPower, SpellCritDamage
- **Blood:** PhysicalResistance, SpellResistance, SpellCooldownRecovery, MinionDamage

## Shift Spells

Each class has themed spells that unlock through prestige:

- Default spell (Veil of Shadow) available to all classes
- Additional spells unlock at prestige levels 0, 1, 2, 3, 4, 5

Use `.class csp [#]` to select which unlocked spell to use on Shift.

## Chat Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| `.class list` | `.class l` | List available classes |
| `.class select [Class]` | `.class s` | Select a class (free first time) |
| `.class change [Class]` | `.class c` | Change class (costs items) |
| `.class listspells [Class]` | `.class lsp` | Show class spells |
| `.class liststats [Class]` | `.class lst` | Show class synergies |
| `.class choosespell [#]` | `.class csp` | Select shift spell |
| `.class lockshift` | `.class shift` | Toggle shift spell |

## Configuration

### Core Settings

| Option | Default | Description |
|--------|---------|-------------|
| `ClassSystem` | `false` | Enable/disable classes |
| `ClassOnHitEffects` | `true` | Enable on-hit procs |
| `OnHitProcChance` | `0.075` | Proc chance (7.5%) |
| `SynergyMultiplier` | `1.5` | Synergy stat multiplier |

### Class Change Cost

| Option | Default | Description |
|--------|---------|-------------|
| `ChangeClassItem` | `576389135` | Item PrefabGUID for change |
| `ChangeClassQuantity` | `750` | Cost to change class |

### Spell Configuration

Each class has configurable spell lists. See [Configuration Reference](/reference/config) for details.
