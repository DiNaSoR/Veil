# Client (EclipsePlus)

EclipsePlus is the client-side mod that provides the UI overlay for BloodCraftPlus.

## Overview

EclipsePlus displays all your progression data through:
- **HUD Components** — On-screen progress bars
- **Character Menu** — Full-featured tabbed interface
- **Real-time sync** — Data updated from server

<Callout type="warning">
  EclipsePlus requires the Bloodcraftplus server mod to function!
</Callout>

---

## Key Components

### HUD Components

Real-time progress bars displayed on screen:

| Component | Purpose |
|-----------|---------|
| **Experience Bar** | Player XP progress |
| **Expertise Bar** | Weapon mastery progress |
| **Familiar Bar** | Active familiar health/level |
| **Legacy Bar** | Blood legacy progress |
| **Quest Tracker** | Active quest objectives |

### Character Menu

Press **F8** to open the full-featured Character Menu:

| Tab | Contents |
|-----|----------|
| **Class** | Class selection and spells |
| **Exoform** | Exoform transformation UI |
| **Familiars** | Full familiar management |
| **Prestige** | Prestige leaderboards |
| **Professions** | Profession levels |
| **Progression** | Combined overview |
| **Stat Bonuses** | Expertise/Legacy stats |

## Data Flow

Server → Client data sync:

1. **Server generates events** — Kill, craft, level up, etc.
2. **Progression updates** — Server calculates new values
3. **Chat protocol** — Data encoded and sent to client
4. **DataService parses** — Client extracts and caches data
5. **UI renders** — HUD and menus update from cache

See [Data Flow](/client/data-flow) for detailed architecture.

## Customization

### HUD Layout

Use the Layout Editor (F9) to:
- Drag HUD elements to new positions
- Save layout preferences
- Reset to defaults

### Localization

EclipsePlus supports multiple languages:
- Prefab names localized based on server setting
- UI text matches game language
