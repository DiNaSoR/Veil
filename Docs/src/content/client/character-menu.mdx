# Character Menu

The Character Menu is the main UI for viewing and managing your BloodCraftPlus progression.

## Overview

Press **F8** to open the Character Menu. It provides:
- Tabbed navigation between systems
- Detailed progression views
- Interactive actions (no chat typing needed)

## Navigation

### Main Tabs

The top row shows major sections:
- Click to switch between tabs
- Active tab is highlighted

### Sub-Panels

Some tabs have internal modes:
- **Stat Bonuses:** Weapon / Blood toggle
- **Familiars:** Familiars / Battle Groups toggle

---

## Interactions

### Click Actions

Most interactions are click-based:
- Select items from lists
- Toggle options on/off
- Confirm actions

<Callout type="tip">
  Hover over elements to see tooltips and additional info.
</Callout>

### Chat Commands

Some actions still require chat (for safety):
- Prestige actions
- Permanent deletions
- Admin commands

## Implementation

### Architecture

```
Services/CharacterMenu/
├── Tabs/
│   ├── ClassTab.cs
│   ├── ExoformTab.cs
│   ├── FamiliarsTab.cs          # Main file
│   ├── FamiliarsTab.Battles.cs  # Battle groups partial
│   ├── FamiliarsTab.Overflow.cs # Overflow partial
│   ├── FamiliarsTab.Settings.cs # Settings partial
│   ├── FamiliarsTab.Talents.cs  # Talents partial
│   ├── PrestigeTab.cs
│   ├── ProfessionsTab.cs
│   ├── ProgressionTab.cs
│   └── StatBonusesTab.cs
├── Shared/
│   └── UIFactory.cs             # Shared UI builders
├── Base/
│   └── CharacterMenuTabBase.cs  # Tab base class
└── CharacterMenuIntegration.cs  # Tab registration
```

### Tab Registration

Tabs implement `ICharacterMenuTab` interface:
- `Build()` — Create UI structure
- `Update()` — Refresh data
- `SetActive()` — Show/hide handling

---

## Data Binding

Character Menu reads from `DataService`:
- Cached server data
- Parsed progression states
- Real-time updates on sync
