# Installation Guide

This guide walks you through installing BloodCraftPlus for both clients and servers.

## Prerequisites

### BepInEx Installation

BloodCraftPlus requires BepInEx 6.x (IL2CPP version). If you don't have BepInEx installed:

1. Download **BepInEx 6.x** for IL2CPP from the [BepInEx releases](https://github.com/BepInEx/BepInEx/releases)
2. Extract to your V Rising or V Rising Server folder
3. Run the game once to generate the `BepInEx/plugins` folder

<Callout type="danger">
  Do NOT use BepInEx 5.x (Mono) â€” V Rising requires the IL2CPP version!
</Callout>
