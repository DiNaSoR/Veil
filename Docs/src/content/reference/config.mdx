# Configuration Reference

All server configuration options for Bloodcraftplus.

<Callout type="info">
  Configuration file location: `BepInEx/config/com.dinasor.bloodcraftplus.cfg`
</Callout>

---

## General

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `LanguageLocalization` | string | `"English"` | Language for prefab names. Options: Brazilian, English, French, German, Hungarian, Italian, Japanese, Koreana, Latam, Polish, Russian, SimplifiedChinese, Spanish, TraditionalChinese, Thai, Turkish, Vietnamese |
| `Eclipsed` | bool | `false` | Controls sync frequency. true = fast (0.1s), false = slow (2.5s) |
| `ElitePrimalRifts` | bool | `false` | Enable elite primal rifts |
| `RiftFrequency` | int | `6` | Rifts per day (max 24). Set to `0` to disable rifts. |
| `EliteShardBearers` | bool | `false` | Enable elite shard bearers |
| `ShardBearerLevel` | int | `0` | Shard bearer level (0 = no effect) |
| `PotionStacking` | bool | `false` | Allow t01/t02 potion effects together |
| `BearFormDash` | bool | `false` | Enable bear form dash |
| `BleedingEdge` | string | `""` | Weapon-specific changes (Slashers, Crossbow, Pistols, TwinBlades, Daggers) |
| `TwilightArsenal` | bool | `false` | Experimental shadow weapon abilities |
| `PrimalJewelCost` | int | `-77477508` | Item prefab for primal jewel crafting |

<Callout type="warning">
  Some options are cached at server startup and require a **server restart** to apply changes
  (for example: `ElitePrimalRifts`).
</Callout>
